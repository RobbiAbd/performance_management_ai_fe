# SIPALING FE - Project Summary

## 📋 Overview

**SIPALING** (Sistem Informasi Pangan dan Bahan Pokok Nasional) adalah aplikasi frontend untuk sistem informasi ketahanan pangan nasional yang menggunakan teknologi Machine Learning untuk clustering dan analisis data.

## 🎨 Design System

Aplikasi ini menggunakan **Neobrutalism Design** dengan karakteristik:
- Bold borders (border-2)
- Shadow effects (shadow-shadow)
- Rounded corners (rounded-base)
- Bright accent colors
- Bold typography (font-heading)
- Interactive hover effects

## 🚀 Fitur yang Telah Dibuat

1. **Routing System** - React Router DOM v7.9.6
2. **Navigation Bar** - Sticky navbar dengan active route highlighting
3. **Landing Page** - Hero section, features, dan CTA
4. **About Page** - Informasi tentang SIPALING
5. **Neobrutalism UI Components** - 17 UI components siap pakai

## 📦 Dependencies Utama

- React 19.2.0
- React Router DOM 7.9.6
- Tailwind CSS 4.1.17
- Radix UI Components
- Lucide React (Icons)
- Recharts (Charts)
